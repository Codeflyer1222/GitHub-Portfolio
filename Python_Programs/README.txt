You must install the PIL module in Python in order to run the game. The game is still buggy right now, but the core of the word game should still work.
To play the game, you must guess as many words as possible with the given letters. Every correct answer is rewarded with a point and a happy meme. An incorrect
answer is punished with a sad meme.

You should only need to download the zip file, which contains only the important files for running the game.
